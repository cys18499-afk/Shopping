# Shopping Mall Project

상품 조회, 검색, 리뷰, 장바구니, 위시리스트, 주문 기능을 제공하는 쇼핑몰 웹 서비스입니다.

회원 및 비회원 주문을 지원하며, 마이페이지를 통해 주문 내역, 배송지, 회원 정보를 관리할 수 있습니다.

## 주요 기술 구현

### Zustand 기반 전역 상태 관리

- AuthStore, CartStore, WishStore, OrderStore를 통한 유기적인 상태 구조 설계
- 비회원 사용자: Zustand Persist 미들웨어를 적용하여 브라우저(로컬 스토리지)에 장바구니 상태 유지
- 로그인 사용자: Supabase 백엔드와 실시간 장바구니 데이터를 동기화하여 멀티 디바이스 환경 대응
- 토스페이먼츠 결제 위젯 및 적립금 적용을 위한 OrderStore 기반의 결제 흐름 제어

### 성능 최적화

- ISR을 활용한 상품 및 카테고리 페이지 캐싱
- React Query를 활용한 서버 상태 캐싱
- Suspense 기반 부분 로딩 처리

### 데이터 관리

- Supabase를 활용한 인증 및 데이터 관리
- React Query를 활용한 서버 상태 관리
- Zustand를 활용한 클라이언트 상태 관리

## 트러블 슈팅

### 무한 스크롤 중복 요청 방지

#### 문제

- Intersection Observer의 콜백이 연속으로 호출되면서, 이전 요청이 끝나기 전에 동일한 페이지 데이터에 대한 중복 API 요청 발생

#### 해결

- useRef를 활용하여 현재 API 요청이 진행 중인지를 판별하는 상태 플래그를 구현하고, 요청이 완료되기 전까지는 추가적인 스크롤 이벤트 호출을 전면 차단

#### 결과

- 불필요한 네트워크 트래픽 감소 및 데이터 중복 렌더링 오류 완벽 차단

---

### 낙관적 업데이트 적용

#### 문제

- 서버 응답 대기 중 UI 반영 지연

#### 해결

- 상태를 먼저 반영 후 서버 동기화
- 실패 시 이전 상태로 롤백

#### 결과

- 즉각적인 UI 반응 제공
- 사용자 경험 향상

---

### 로딩 UI 최적화

#### 문제

- 짧은 요청에도 전체 로딩 화면 노출

#### 해결

#### 결과

- 화면 깜빡임 감소
- 초기 렌더링 경험 개선
